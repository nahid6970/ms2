** Examples
#+BEGIN_SRC 

Hides the word if before there is a \.



Monospace (Code):
Use equal signs = or ~ before and after the text you want to format as monospace.
Example: =Monospace Text=
Example: ~Monospace Text~

Superscript:
Use caret ^ before word you want to make superscript.
Example: ^Superscript

Subscript:
Use underscores _ before the word you want to make subscript.
Example: _Subscript

[[URL][description]]       --hyperlinks
\textcolor{red}{Text}      --colored text
1., 2., etc.,              --for ordered lists
- or +                      --for unordered lists.

[[file:path/to/image.png]]
[[#custom-id][Link to Section]].
[[fn:number]]   --footnote



| ORG Functionality  | Sign   | Example         | Details / Usage                  |
|--------------------+--------+-----------------+----------------------------------|
| Bold               | **     | **Hello World** | ** before and after text         |
| Check / Uncheck    | - [ ]  | - [ ] Not Done  |                                  |
| Check / Uncheck    | - [X]  | - [X] Done      |                                  |
| Code               | = or ~ | =Hello World=   | = or ~ befor and after text      |
| Display Math Mode: | $$     | $$E=mc^2$$      | middle                           |
| Horizontal Rule    | -----  | -----           | ----- in a line without any text |
| Inline Math Mode:  | $      | $E=mc^2$        | left side                        |
| Italicize          | /      | /Hello World/   | / before and after text          |
| Strikethrough      | +      | +Hello World+   | + before and after text          |
| Underline          | _      | _Hello World_   | _ before and after text          |
| Hide Word          | \      | \This           | \ before the word                |

#+END_SRC
